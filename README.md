🔐 Caesar Cipher (Encrypt & Decrypt Tool)


A simple Python program that encrypts and decrypts messages using the Caesar Cipher technique by shifting letters in the alphabet.

🚀 Features...
Encrypt text using letter shifting
Decrypt messages back to original form
Supports full alphabet (a–z)
Keeps spaces and special characters unchanged
Option to run again

💻 Tech Used...
Python 3

String manipulation
Functions in Python
Basic cryptography logic
Modular arithmetic (%)
